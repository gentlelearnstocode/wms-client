# wms-client

## Description

wms-client is the frontend repository to serve wms-service restful api @ [wms-service](https://github.com/gentlelearnstocode/wms-service)

## Requirements

- node >= 18.16.0
- vite >= 4.1.0

## Installation

- To install run: `npm install`
- Run the app: `npm run start-dev`
